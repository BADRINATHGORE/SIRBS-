🚆 Smart Indian Rail Booking System

Smart Indian Rail Booking System with Admin & User Modules, Ticket Booking, Train Management, and Fare Inquiry.

Project Overview

This system modernizes railway booking using Java, JSP, and Oracle Database. Users can search trains, book tickets, check fares, and manage profiles, while admin handles complete system operations.
